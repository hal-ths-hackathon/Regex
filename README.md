# 💣 Regex(レジェックス)エスケープ

「正規表現」の記号の羅列を解読し、時限爆弾を解除せよ！
視覚的なフィードバックで、遊びながら正規表現が身につく脱出パズルゲームです。

## ✨ コアバリュー（このアプリの魅力）
* **リアルタイム・ハイライト**: 自分の入力した正規表現が、ノイズ文字列の中から「何を捕まえているか」が瞬時に視覚化されます。
* **無限の爆弾（プロシージャル生成）**: 問題と答えはシステムが自動生成。挑むたびに異なるパスワードが要求されます。
* **選べる難易度**: 初心者向けの「3択選択モード」と、実力者向けの「直接入力モード」を搭載。

## 🛠 技術スタック
* **Frontend**: React (TypeScript)
* **Backend**: Python (FastAPI)
* **Database**: PostgreSQL
* **Infrastructure**: Docker / Docker Compose
