# プロダクト概要 (Overview)

## ターゲットユーザー
* 正規表現（`\d`, `^`, `$` など）を見ると頭が痛くなるが、業務で必要性を感じている人
* パズルゲーム感覚でプログラミングの基礎を学びたい人

## コア機能と画面フロー
1. **タイトル画面**: モード選択（「3択で解除（Easy）」or「入力で解除（Hard）」）
2. **プレイ画面（時限爆弾）**:
    * 制限時間のタイマーがカウントダウン
    * 大量のダミー文字列（ノイズ）が表示される
    * プレイヤーの入力に応じて、ノイズ内の対象文字列がリアルタイムにハイライトされる
3. **リザルト画面**: 
    * 成功：扉が開き、解除成功タイムを表示。アーケード風に名前（3文字など）を入力してランキング登録。
    * 失敗：爆発エフェクト。
