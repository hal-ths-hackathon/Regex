### 背景
ユーザーが入力した正規表現に応じてリアルタイムで対象文字列を強調し、正解判定までつなげる必要がある。ここがゲーム体験の中心になるため、入力中のエラー処理も含めて安定した挙動にしておく必要がある。

### 実施内容
- [ ] ユーザー入力の正規表現をリアルタイム評価する処理の作成
- [ ] `noise_text` にマッチした箇所をハイライトする処理の作成
- [ ] `correct_string` と完全一致した場合のクリア判定実装
- [ ] 正規表現エラー時の `try-catch` ハンドリング実装
- [ ] エラー時の視覚フィードバック案の整理

### 完了条件
